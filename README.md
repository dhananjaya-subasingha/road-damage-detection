# road-damage-detection
# Road Damage Detection using Image Processing

## Description
This project detects road damages such as cracks, potholes, and road markings using image processing techniques.

## Features
- Image enhancement (histogram equalization + blur)
- Segmentation using thresholding and edge detection
- Morphological processing
- Contour detection and filtering
- Output video generation

## Requirements
- Python
- OpenCV
- NumPy

## How to Run
1. Place input frames/video in input folder
2. Run:
   python main.py
3. Outputs will be saved in output folder

## Outputs
- Enhanced frames
- Segmentation masks
- Final marked images
- Output video
